# RepairShop

Specialized workshop within an MRO (e.g., engine shop, avionics shop).

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| mro | MRO | Reference | 1 |
| name | string | Shop name [LABEL] | Engine Shop 1 |
| specialization | string | Area of specialization | CFM56 Overhaul |
