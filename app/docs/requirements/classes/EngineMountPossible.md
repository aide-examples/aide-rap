# EngineMountPossible

Maps which engine types can be installed on which aircraft types. Could also be named *EngineCompatibility*.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| aircraft_type | AircraftType | Reference [UK1] [LABEL] | 1 |
| engine_type | EngineType | Reference [UK1] [LABEL2] | 2 |

## Data Generator

Find out which egine types from the list are allowed to be mounted uder the wings of the listed aircraft types. Create a record for each valid assoiciation.
