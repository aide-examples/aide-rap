# EngineStandCradle

Cradle adapter that holds a specific engine type. Mounted on a base for transport.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| serial_number | string | Cradle serial number [LABEL] | ESC-CFM56-001 |
| engine_type | EngineType | Compatible engine type [LABEL2] | 1 |
| manufacturer | string | Cradle manufacturer | Dedienne Aerospace |

## Data Generator

Create cradles for the most common engine types (CFM56, PW1100G, etc.).
