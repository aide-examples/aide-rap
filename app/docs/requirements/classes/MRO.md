# MRO

Maintenance, Repair and Overhaul organization.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| name | string | MRO name [LABEL] | Lufthansa Technik |
| location | string | Location [LABEL2] | Hamburg |
| country | string | Country | Germany |

## Data Generator

Certified Maintenance and Repair Operators for jet engines which are compatible with commercial aircraft produced by Boeing and Airbus. May include the Engine OEMs themselves
