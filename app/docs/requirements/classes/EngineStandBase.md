# EngineStandBase

Wheeled base frame for transporting engines. Can be combined with different cradles.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| serial_number | string | Base serial number [LABEL] | ESB-2024-001 |
| manufacturer | string | Base manufacturer [LABEL2] | Dedienne Aerospace |
| max_weight_kg | int | Maximum load capacity in kg | 5000 |

## Data Generator

Create 5-10 engine stand bases from manufacturers like Dedienne Aerospace, MTU Aero Engines, or generic suppliers.
