# EngineStandMounting

History of engine placements on transport stands. Tracks which engine was on which stand and when.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| stand | EngineStand | Engine stand [LABEL2] | 1 |
| engine | Engine | Mounted engine [LABEL] | 1 |
| mounted_date | date | Date engine was placed on stand | 2024-01-15 |
| removed_date | date | Date engine was removed (null if still mounted) | null |

## Data Generator

Create mounting records showing engines being placed on stands during shop visits or storage periods.
