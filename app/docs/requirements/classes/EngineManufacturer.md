# EngineManufacturer

Manufacturers of engines (e.g., CFM, Pratt & Whitney, Rolls-Royce).

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| name | string | Company name [LABEL] | CFM International |
| country | string | Country of origin [LABEL2] | USA/France |
| icao_code | string | ICAO identifier | CFM |

## Data Generator

List the largest manufacturers of jet engines for commercial aviation: Only mention companies whose engines can be mounted on Airbus or Boeing airraft.
