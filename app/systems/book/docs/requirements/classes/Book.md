# Book

Stores information about published books.

| Attribute | Type | Description | Example |
|-----------|------|-------------|---------|
| isbn | string | ISBN number [LABEL] | 978-0451524935 |
| title | string | Book title [LABEL2] | 1984 |
| publication_year | integer | Year of publication | 1949 |
| genre | string | Literary genre | Dystopian Fiction |
| author | Author | Author of the book | 1 |

## Data Generator

Create 2-3 books for each author in the database.
